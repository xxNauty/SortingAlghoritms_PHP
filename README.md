# Algorytmy Sortowania w PHP

Zbiór algorytmów sortowania zaimplementowanych w PHP.

## Funkcje
- Implementacje popularnych algorytmów sortowania.
- Napisane w całości w PHP.
- Przejrzysty i zwięzły kod w celach edukacyjnych.
